# Focus Questions

1. 哪些 ECM 成分或合成材料最常用于支持特定类器官的形成与成熟？
2. 机械性质如 stiffness、viscoelasticity、degradability 如何影响 organoid morphology 和 function？
3. 合成或半合成 ECM 相比 Matrigel 在可重复性和临床转化上有哪些优势与限制？
4. 哪些 readout 最常用于衡量 ECM 对类器官合成的影响，例如 size、budding、polarity、marker expression、single-cell profile？
5. 当前证据中最明显的空白是什么：材料设计、机制解释、标准化、还是特定器官系统的验证不足？
